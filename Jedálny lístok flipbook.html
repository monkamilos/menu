🧩 1. Použiť „flipbook“ plugin alebo knižnicu

Toto je najefektívnejší spôsob pre efekt „otáčania strán ako v knihe“:

✅ Riešenia:

  • WordPress plugin:

    • Prehliadač , FlipBook ] ( https : / /3D FlipBook – prehliadač PDF, FlipBook

    • DearFlip –– Jednoduchý flipbook pre PDF alebo obrázky.

  • Pre vlastný web (HTML/JS):

    • – populárna JavaScript knižTurn.js – populárna JavaScript knižnica pre efektné otáčanie strán.

    • umožňuje FlipHTML5 – online nástroj na tvorbu flipbookov (umožňuje embed na web).

💡 Ako to funguje:

  1. Nahraješ jedálny lístok ako PDF alebo salebo sadu obrázkov .

  2. Pomocou pluginu alebo kódu vložíte na web tzv. komponent „flipbook“.

  3. Výsledok: návštevník listuje stránkami ako v knihe (aj s animáciou).

🔧 2. Vlastné HTML + CSS riešenie

Ak nechcete používať externé nástroje alebo pluginy, môžete si urobiť vlastný jednoduchý listovací efekt.

  • Môžete použiť karusely (napr. Swiper.js) alebo vlastnú logiku na preklikávanie strán.

  • Nezískaš animáciu ako pri skutočnom knižnom liste, ale vieš to urobiť rýchlo a responzívne .

📝 Príklad s Turn.js (ako web HTML):

html

</div>

<script src="https://cdnjs.cloudflare.com/ajax/libs/turn.js/4.1.0/turn.min.js"></script>

    width: 800,
    height: 600,
    autoCenter: true
  });
</script>
